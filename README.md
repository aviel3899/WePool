# WePool 🚗

An internal carpool management application for organizations, developed as a graduation project at Afeka College.

WePool enables employees from the same organization to coordinate safe, reliable, and efficient shared rides.
By utilizing an intelligent matching engine based on preferred arrival times, allowed route deviations, and available seats, the app promotes trust, enhances security, and improves commuting experiences within a corporate environment.

---

## 🚀 Features

- 🔑 Role-based system: Driver, Passenger, HR Manager, and Admin.
- 📍 Smart ride-matching algorithm using real-time route analysis.
- 🗺️ Integration with Google Maps APIs (Directions, Places, Route Decoding).
- 🔥 Backend powered by Firebase (Authentication, Firestore, Storage).
- 📱 Full mobile app UI developed with Kotlin and Jetpack Compose.
- 🧠 Dynamic ride management.
- ✅ Secure user authentication and role authorization.

---
